# cpp_onboarding

## Project Overview

This repository is a compilation of smaller C++ projects where the goal was to gradually learn new concepts along the way, starting all the way from how the syntax differs from C and how to use classes efficiently.

## Concepts Learned

- C++ syntax basics: classes, member functions, constructors
- Stack vs heap allocated memory
- Using pointers vs references
- Orthodox Canonical Class form
- Operator overloading
- Inheritance, polymorphism & abstract classes
- Different types of casts
- Exceptions
- Templates

## Learnings

- This journey felt quite valuable coming in with only C experience. C++ felt very flexible and efficient compared to C, which oftentimes feels much more granular. Although that freedom introduced new complexity too as it took quite some time to understand that the standard library actually offers a lot of functionality, for example when it comes to string manipulation. So there was always a sense that perhaps there may be better ways to handle things compared to your own implementation. But all in all, I enjoyed learning C++ a lot and I was especially grateful to learn about the concept of throwing exceptions.

## How to Run the Project

- Clone the repository, navigate to the program you want to run, and compile it with the provided Makefile.
